# TerraGuard - Soil Erosion Detection

An AI-powered tool for SDG 15 (Life on Land) that detects soil erosion from images and provides localized remediation guides.

## Features

- Modern Streamlit UI with green/earthy color palette
- Image upload for JPG/PNG files
- Simulated AI detection of soil erosion states
- Remediation action plans with step-by-step instructions
- PDF report generation
- Intel OpenVINO optimization placeholder

## Tech Stack

- Python
- Streamlit
- OpenCV
- PIL (Pillow)
- ReportLab

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

Upload an image of land/fields to analyze erosion and get remediation recommendations.

## Project Structure

- `app.py`: Main Streamlit application
- `requirements.txt`: Python dependencies
- `README.md`: Project documentation
